Mandatory session end protocol. Run this before closing every session.

## Steps

### Step 1 — Update MEMORY.md
Read the current `MEMORY.md` (project root), then update:

- **## Last Session Summary** — Add a dated entry describing what was done, what worked, what broke
- **## Current State** — Update file counts and system inventory if files were added/changed
- **## Known Bugs** — Add new bugs with file+line. Remove bugs that were fixed.
- **## Next Up** — Reprioritize based on what was just completed
- **## Decisions Made** — Add any new architecture decisions with reasoning
- **## What NOT To Do** — Add any failed approaches discovered this session

### Step 2 — Update auto-memory
Run `/update-memory` to sync the auto-memory files.

### Step 3 — Confirm
State in one sentence what MEMORY.md now says the next session should start with.

A session without a memory update is a session that will be repeated.
